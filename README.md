# mockk-example
코틀린 언어의 테스트 도구인 https://mockk.io 기능을 샘플코드로 작성해본다.

### Goal 
- Mockk를 처음 접하는 동료들에게 샘플코드를 통해 직접적인 도움
- 샘플 코드를 통한 mockk 이해
- 원하는 기능에 대해서 Copy & Paste가 가능 수준 제공
